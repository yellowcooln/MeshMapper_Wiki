# 🚀 MeshMapper Update (March 4, 2026)

### ✨ New Features

* **Effective Coverage Map** — A new map layer that visualizes coverage based on effective reach! *(Thanks HerculesMulligan)*

![Image](https://meshmapper.net/img/upload_20260304_174704_3564bc42.jpg)

* **Signal Strength Map** — A new map layer that color-codes coverage by signal strength (SNR), making it easy to identify strong and weak zones at a glance. *(Thanks davidjay)*

![Image](https://meshmapper.net/img/upload_20260304_175903_005745e0.jpg)

* **In/Out Boundary Labels on Repeaters** — The repeaters table in the regional admin panel now indicates whether a repeater is inside or outside the region boundary. *(Thanks Michał Konopiński)*

![Image](https://meshmapper.net/img/upload_20260304_174827_5be0a2aa.jpg)

* **Public Repeater ID List** — A publicly accessible list of a regions repeater ID utilization is now available. Find this on the "Region Info" page of a region and click "Repeater List".  *(Thanks Michał Konopiński)*

![Image](https://meshmapper.net/img/upload_20260304_174906_a23bf233.jpg)

* **Bulk Select & Batch Edit/Delete** — Regional administrators can now multi-select repeaters and companions in the admin panel for batch editing or batch deletion. *(Thanks davidjay)*

![Image](https://meshmapper.net/img/upload_20260304_175032_c27bb332.jpg)

* **Ping Reassociation Tool** — A new maintenance tool in the admin panel allows administrators to reassociate pings to a repeater that was missing from the map at the time of ingestion. The tool analyzes affected pings, detects ID collisions, and handles both standard and DISC pings. *(Thanks NoahK)*

![Image](https://meshmapper.net/img/upload_20260304_175153_4646198d.jpg)

* **Admin Contact Info** — Administrators can now update their contact information directly from the User Settings tab in the admin panel. *(Thanks AI7NC)*

![Image](https://meshmapper.net/img/upload_20260304_175248_76a0cce3.jpg)

* **Grant Existing Admins Access to New Regions** — When adding a new administrator, if the administrator already exists, access to that region in question will be given to that existing administrator instead of having to create a new account. *(Thanks AI7NC)*

![Image](https://meshmapper.net/img/upload_20260304_175419_7d5f4c4a.jpg)

### 🔧 Improvements

* **Repeater Neighbours Fade Effect** — When viewing Repeater Neighbour lines, coverage pings now fade into the background to make the neighbour connections more visible and easier to trace. *(Thanks EldoonNemar)*

![Image](https://meshmapper.net/img/upload_20260304_175611_dcd93110.jpg)

### 🐛 Bug Fixes

* Fixed an issue where search filters, sort order, and pagination were reset after deleting a repeater, companion, or other record in the admin panel. State is now preserved across edit and delete operations. *(Thanks davidjay)*
