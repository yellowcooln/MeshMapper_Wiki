🔧 **MeshMapper v1.1.1 [HOTFIX]**

🐛 **Bug Fix**

- Fixed a scope matching issue that caused pings to be sent with the wrong packet type. Zones without a regional scope use `#*` as a wildcard, but the app only recognized `*`, which caused it to set the wrong flood scope on the radio. This meant TX pings went out as TransportFlood instead of regular FLOOD.

⚙️ **Improvements**

- Debug logs are now enabled by default

- Log uploads now prompt you to describe what's going on before submitting, making it easier for us to diagnose issues. The upload button has also moved beside "Delete All" under Debug Logs.
