# Wardriving App

MeshMapper is a community-driven wardriving app for MeshCore mesh network devices. It lets you connect to a MeshCore radio over Bluetooth, send GPS-tagged pings across the mesh, and contribute coverage data to the [MeshMapper community map](https://meshmapper.net/).

The app is available on Android, iOS, and Chromium-based browsers (Chrome/Edge). The source code is available on [GitHub](https://github.com/MeshMapper/MeshMapper_Flutter_App).
